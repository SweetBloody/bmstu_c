# Тесты для лаб. раб. №7

## Входные данные
Файл с целыми числами

## Выходные данные
Файл с целыми числами

## Позитивные тесты:
- 01 - Сортировка неупорядоченного массива;
- 02 - Сортировка упорядоченного по возрастаниию массива;
- 03 - Сортировка упорядоченного по убыванию массива;
- 04 - Сортировка массива из одинаковых чисел;
- 05 - Фильтрация массива с 1 отрицательным элементом, не являющимся первым;
- 06 - Фильтрация массива с 2 отрицательными элементами, не являющимися первыми;
- 07 - Фильтрация массива без отрицательных элементов;

## Негативные тесты:
- 01 - Несуществующий файл;
- 02 - Пустой файл;
- 03 - В командной строке 2 аргумента;
- 04 - В командной строке 5 аргументов;
- 05 - 4 параметр командной строки - не буква f;
- 06 - Во входном файле находятся не только числа;
- 07 - Фильтрация массива: первое число - отрицательное;
